# windows激活
1. 右键开始图标，选择“Windows PowerShell(管理员)”
2. 复制以下命令到Powershell窗口, 回车
``` powershell
irm https://get.activated.win | iex
```
按1（数字许可证激活，HWID方式，永久激活Windows）
